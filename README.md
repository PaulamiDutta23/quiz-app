# Quiz App

## Objective:

This is an assignment to create an end to end quiz app using Vanilla JS. Our
moto is to provide questions one after another and user will answer those. On
submit the next question will show up. At the end we have to show the report
card on submission of the final question.

## Learnings:

- End to end creation of an app
- Mock the frontend
- Start a project with frontend
- Develop small things to achieve the long term goal
- Do not break the working things
- Think of small tasks (break a problem statement into sub parts)

## Future Plans:

- Make authentication of user login-logout
- Show the remaining number of question
- Provide indication for the correct/incorect answer
- Save progress of the users (name, scores, rank)
- Generate the questions randomly
- Scramble the options
- Provide negative score for wrong answers and positive score for right answer
